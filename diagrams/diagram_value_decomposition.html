Value Decomposition & IGM — From VDN to QMIX
Predecessor
VDN
Linear decomposition: Q_tot is the sum of individual utilities. Simple, guaranteed to satisfy IGM, but too restrictive for non-linear agent interactions.
Q_tot = Q₁(s₁,a₁) + Q₂(s₂,a₂)
Linear · satisfies IGM
but limited expressivity
This Implementation ✓
QMIX
Arbitrary monotonic mixing: Q_tot can be any function of Q₁, Q₂ as long as it is strictly increasing in each argument. Non-negative mixing weights enforce this.
Q_tot = f_mix(Q₁, Q₂; z)
subject to: ∂Q_tot/∂Qᵢ > 0
Curved surface · IGM guaranteed ✓
Violation — What Breaks IGM
Non-Monotonic
If the mixing function decreases in any Qᵢ direction, greedy local action selection no longer produces the global optimum. IGM fails — joint optimization is required.
∃ Qᵢ: ∂Q_tot/∂Qᵢ < 0
IGM breaks · can't execute
greedily → correct

Non-negative activation derivative everywhere
Step 3 · Chain rule
∂Q_tot/∂Qᵢ = W₂ · diag(ELU') · W₁[:,i]
Product of three non-negative quantities
